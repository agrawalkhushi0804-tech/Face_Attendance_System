# 🎯 Face Recognition Attendance System

A Python-based Face Recognition Attendance System that automates attendance marking using real-time facial recognition. The application captures live video through a webcam, recognizes registered users using DeepFace, and records attendance with the current date and time in a CSV file.

## ✨ Features

- Face registration with image capture
- Real-time face recognition using a webcam
- Automatic attendance recording
- Attendance stored in CSV format
- Duplicate attendance prevention
- User-friendly GUI built with CustomTkinter

## 🛠️ Technologies Used

- Python
- OpenCV
- DeepFace
- CustomTkinter
- Tkinter
- Pandas
- NumPy
